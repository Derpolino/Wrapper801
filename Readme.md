# Wrapper801
## Introduction
Wrapper801 est un module Python permettant à tous de créer son propre bot/service utilisant le forum d'[atelier801].

[atelier801]: http://atelier801.com
****
## Utilisation
Initialisation :
```py
from a801py import *
bot = wrapper("Username", "Password")
```
Connection :
```py
bot.connect()
```
